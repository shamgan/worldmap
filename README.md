# מפה אינטראקטיבית — מדדים דמוגרפיים וסוציו-אקונומיים בעולם

מפה אינטראקטיבית (Leaflet) המציגה 9 שכבות מידע עבור 215 מדינות, עם ציר זמן היסטורי (1975–2024):

1. **מאזן הגירה** — `SM.POP.NETM`
2. **תוחלת חיים** — `SP.DYN.LE00.IN`
3. **אחוז יודעי קרוא וכתוב** — `SE.ADT.LITR.ZS` (World Bank/UNESCO) + השלמות מ-CIA World Factbook ארכיוני
4. **שיעור ילודה** — `SP.DYN.CBRT.IN`
5. **שיעור תמותה** — `SP.DYN.CDRT.IN`
6. **אחוז עירוניות** — `SP.URB.TOTL.IN.ZS`
7. **אוריינות פונקציונלית (OECD PIAAC)** — שכבה משלימה ל-#3, ~32 מדינות בלבד (ראו הסבר מתודולוגי למטה)
8. **גידול טבעי (מחושב)** — שיעור ילודה פחות שיעור תמותה, מחושב מתוך שכבות #4 ו-#5
9. **אוכלוסייה** — `SP.POP.TOTL`

לכל שכבה מקרא צבעים (choropleth) עם 5 קטגוריות, ולכל מדינה מוצג הערך העדכני ביותר הזמין
במאגר **World Bank Open Data**, יחד עם שנת הנתון (השנים שונות ממדינה למדינה, בהתאם לזמינות הדיווח).

### יכולות אינטראקטיביות

- **ציר זמן היסטורי (1975–2024)** — לשכבות #1,2,4,5,6,8,9: החלקה בין שנים או הפעלה אוטומטית, עם נתונים אמיתיים מהבנק העולמי לכל שנה (לא רק הערך העדכני)
- **חיפוש מדינה** עם autocomplete, קפיצה ישירה במפה
- **מיקום בעולם / אחוזון** — לכל מדינה, לפי השכבה הפעילה (ולפי השנה הנבחרת בציר הזמן)
- **דירוג מדינות** — top-10/bottom-10 לשכבה הנוכחית, עם קפיצה למדינה בלחיצה
- **סינון לפי אזור/יבשת** — מדגיש מדינה אחת ומעמעם את השאר
- **קישור לשיתוף** — השכבה, המדינה הנבחרת והסינון נשמרים בכתובת ה-URL וניתנים לשליחה
- **השוואת מדינות** — טבלה צד-לצד של שתי מדינות נבחרות על פני כל השכבות
- **טבלת נתונים** — כל 215 המדינות, ניתנת למיון לפי כל עמודה ולסינון לפי שם
- **השוואת שכבות (Swipe)** — שתי שכבות שונות זו לצד זו על אותה מפה, עם קו גרירה
- **מפה דו-משתנית (Bivariate)** — שילוב חזותי של שתי שכבות במקביל, רשת צבעים 3×3
- **שכבות בסיס** — כהה / בהיר (CARTO), טופוגרפית (OpenTopoMap), תאורת לילה מלוויין (NASA VIIRS Black Marble)
- **קו יום/לילה בזמן אמת** — טרמינטור מחושב מקומית בדפדפן (ללא שירות חיצוני), מציג את קו הגבול הנוכחי בין יום ולילה על כדור הארץ
- **מצב תצוגה בהיר/כהה** — נשמר בדפדפן

🔗 דמו: פתחו את `index.html` (עובד גם בלחיצה כפולה ישירה מהדיסק — הנתונים משובצים בקובץ עצמו),
או את הכתובת של GitHub Pages לאחר פרסום.

## מבנה הפרויקט

```
index.html                    # האפליקציה — עמוד יחיד. הנתונים משובצים ישירות בקובץ (ראו למטה)
data/
  world_data.geojson          # גבולות מדינות + כל הנתונים המספריים (עותק זהה למשובץ ב-index.html)
  indicator_stats.json        # סטטיסטיקות (מינ/מקס/קוונטילים) לכל שכבה, לצורך המקרא
sources/                      # קבצי המקור הגולמיים + סקריפט העיבוד (לשחזור/עדכון הנתונים)
  process_data.py
  hebrew_names.py             # מיפוי קוד ISO3 -> שם מדינה בעברית
  piaac_data.py                # נתוני OECD PIAAC (אוריינות פונקציונלית) לפי ISO3
  legacy_literacy_data.py      # השלמות אחוז יודעי קרוא וכתוב מ-CIA World Factbook ארכיוני, לפי ISO3
  countries.geojson           # גבולות מקור (Natural Earth, ברזולוציה גבוהה)
  countries_meta.json         # רשימת מדינות מהבנק העולמי (לסינון אזורים מצרפיים)
  wb_*.json                   # תשובות API גולמיות מהבנק העולמי (ערך עדכני), אינדיקטור לכל קובץ
  wb_hist_*.json               # תשובות API גולמיות מהבנק העולמי (סדרת 1975–2024), לכל אינדיקטור
```

**הערה טכנית**: `index.html` טוען את הנתונים משני תגי `<script type="application/json">` המשובצים
בתוכו (לא מ-`fetch`), כדי שהקובץ יעבוד גם בלחיצה כפולה ישירה מהדיסק — דפדפנים חוסמים `fetch` לקבצים
מקומיים תחת `file://`. אם התגים ריקים (`{}`) הקוד חוזר אוטומטית ל-`fetch` מתיקיית `data/` (למשל בזמן
פיתוח מול שרת מקומי). `sources/process_data.py` משבץ את הנתונים העדכניים לתוך `index.html` אוטומטית
בסיום כל הרצה.

## מקורות הנתונים

- **נתונים סטטיסטיים**: [World Bank Open Data](https://data.worldbank.org/) — API ציבורי, נשלף ב-2026-07-17.
  לכל מדינה נלקח הערך הלא-ריק העדכני ביותר (`mrnev=1`) עבור כל אינדיקטור. קישורים ישירים לכל אינדיקטור
  מופיעים בחלון "אודות ומקורות" בתוך המפה עצמה.
- **גבולות מדינות**: [datasets/geo-countries](https://github.com/datasets/geo-countries) (מבוסס Natural Earth,
  רמת פירוט Admin-0). הגאומטריה פושטה (Douglas-Peucker, סבילות ~2.2 ק"מ) כדי לשמור על קובץ קל וטעינה מהירה בדפדפן.
- **שמות מדינות בעברית**: תורגמו ידנית ממאגר Hebrew Wikipedia / משרד החוץ (`sources/hebrew_names.py`).
- **אוריינות פונקציונלית (PIAAC)**: [OECD, Survey of Adult Skills, מחזור 2](https://www.oecd.org/en/publications/2024/12/do-adults-have-the-skills-they-need-to-thrive-in-a-changing-world_4396f1f1.html)
  (נתוני שדה 2022–2023, פורסם 2024-12-10). ציון ממוצע באוריינות פונקציונלית, סולם 0–500, ~31 מדינות/כלכלות.

### הערות מתודולוגיות חשובות

- **אחוז יודעי קרוא וכתוב** (`SE.ADT.LITR.ZS`, מקור עיקרי: World Bank/UNESCO Institute for Statistics) חסר
  במקור עבור כ-46 מדינות — בעיקר מדינות עתירות הכנסה (ובהן ארה"ב, קנדה ורוב מערב אירופה) שהפסיקו לשאול על
  אוריינות במפקדי אוכלוסין מזה עשורים, בהנחה שהיא אוניברסלית כמעט לחלוטין. בדקנו ישירות מול ה-API הרשמי של
  UNESCO UIS (כולל מדד הערכה מבוסס-מודל שהם מפרסמים במיוחד למקרים כאלה, "Global Age-Specific Literacy
  Projections") ואין להם כל נתון תואם למדינות האלה.
  **כדי למקסם כיסוי, בהתאם לבקשת המפרסם, השכבה משלבת שני מקורות**: 169 מדינות מ-World Bank/UNESCO (עדכני,
  מהימן), ועוד 20 מדינות המסומנות באייקון ⚠ במפה על בסיס נתון **ארכיוני מ-CIA World Factbook** (האתר נסגר
  ב-2026; המספרים שוחזרו ממראה משנית — indexmundi.com — ומתועדים ב-`sources/legacy_literacy_data.py`).
  נתוני ה-CIA הארכיוניים ישנים מאוד (חלקם מ-1970–2018, לרוב מבוססי "האם ביקר אי-פעם בבית ספר" ולא מבחן
  אוריינות אמיתי) ורמת המהימנות שלהם נמוכה משמעותית — לכן הם מסומנים בבירור בממשק ונבדלים מהמקור העיקרי.
  עבור 8 מדינות/שטחים זעירים בלבד (בעיקר שטחים תלויים כמו קוראסאו, סנט מארטן, סינט מארטן, איי מאן,
  לוקסמבורג, בהאמה, מיקרונזיה ואיי פארו) לא נמצא שום נתון בשום מקור (לא World Bank, לא UNESCO, לא CIA,
  ולא PIAAC) — הם מסומנים "אין נתון".
- כדי לתת מענה איכותי יותר (ולא רק כיסוי) לפער אצל המדינות המפותחות, נוספה גם שכבה נפרדת
  **"אוריינות פונקציונלית (PIAAC)"** — ראו הסבר מלא ב"אודות ומקורות" בתוך המפה. זהו מדד שונה מהותית (ציון
  מבחן בסולם 0–500, לא אחוז יודעי קרוא וכתוב), מכסה כ-32 מדינות בלבד, ואינו ניתן להשוואה ישירה לשכבת "אחוז
  יודעי קרוא וכתוב". עבור בריטניה הנתון מייצג את אנגליה בלבד, ועבור בלגיה — את חבל פלנדריה בלבד, ועבור
  אוסטרליה — נתוני מחזור 1 (2011–2012) ולא מחזור 2 כמו שאר המדינות (ראו `sources/piaac_data.py`).
- **מאזן הגירה** (`SM.POP.NETM`) מדווח על ידי הבנק העולמי כהערכה לתקופה של 5 שנים, ולא כערך שנתי — כך גם
  מוצג במפה.
- כמה מדינות/ישויות זעירות שאינן חלק ממאגר הבנק העולמי (כגון קריית הוותיקן) אינן כלולות במפה, מאחר
  שאין עבורן נתונים להציג.
- **ציר הזמן ההיסטורי** מציג ערכים אמיתיים לכל שנה מהבנק העולמי (לא אינטרפולציה שלנו), אך מקטגר אותם
  לפי טווחי הצבע (המקרא) של הנתון העדכני ביותר בלבד — לכן מדינה יכולה "לצבוע" אחרת בעבר גם אם הצבע לא
  זז בפועל בין שנים סמוכות. שכבות מבוססות-נקודת-זמן-בודדת (PIAAC, וההשלמות הארכיוניות של CIA
  בשכבת יודעי קרוא וכתוב) אינן זמינות בציר הזמן.
- **שכבת "תאורת לילה"** (NASA VIIRS Black Marble) היא תצלום לוויין מצרפי (לא תמונה חיה בזמן אמת) —
  משמש להמחשה ויזואלית של פיזור אוכלוסייה/תשתית, לא כמדד מדעי מדויק. קו ה"יום/לילה" לעומת זאת מחושב
  בזמן אמת לפי השעה הנוכחית בפועל.

## פרסום ב-GitHub Pages

1. צרו מאגר (repository) חדש ב-GitHub והעלו אליו את כל תוכן התיקייה הזו (`index.html`, `data/`,
   `sources/`, `README.md`).
   ```bash
   git init
   git add index.html data README.md sources
   git commit -m "Initial commit: interactive world indicators map"
   git branch -M main
   git remote add origin https://github.com/<username>/<repo-name>.git
   git push -u origin main
   ```
2. במאגר ב-GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch**,
   בחרו `main` ותיקיית `/ (root)`, ושמרו.
3. לאחר דקה-שתיים האתר יהיה זמין בכתובת:
   `https://<username>.github.io/<repo-name>/`

אין צורך בשרת/בילד — האתר הוא HTML/JS סטטי לחלוטין (Leaflet נטען מ-CDN).

## עדכון/שחזור הנתונים

הסקריפט `sources/process_data.py` משחזר את `data/world_data.geojson` ו-`data/indicator_stats.json`
מחדש מהקבצים הגולמיים בתיקיית `sources/`. כדי למשוך נתונים עדכניים מהבנק העולמי:

```bash
cd sources
# למשל, לרענן אינדיקטור בודד:
curl -sL "https://api.worldbank.org/v2/country/all/indicator/SP.DYN.LE00.IN?format=json&per_page=400&mrnev=1" -o wb_SP_DYN_LE00_IN.json
pip install shapely
python process_data.py
```

## טכנולוגיה

עמוד HTML יחיד, ללא תלות ב-build tools. משתמש ב-[Leaflet.js](https://leafletjs.com/) (CDN) לרינדור המפה,
עם שכבת מצע (basemap) כהה מ-[CARTO](https://carto.com/attributions).
